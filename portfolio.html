Iron Ore Price Prediction

  • Technologies Used: Python, Pandas, NumPy, Scikit-learn, TensorFlow/Keras, LSTM, Matplotlib, Seaborn, GPT-J
  • This project focuses on predicting daily iron ore prices using an LSTM-based machine learning model. It integrates multiple data sources, including economic indicators, stock market trends, and sentiment analysis from financial news articles processed with GPT-J. By progressively adding features and optimizing model hyperparameters, the system delivers highly accurate forecasts, providing actionable insights for traders and stakeholders. Water Pump Operation Scheduling

  • Technologies Used: Python, Pandas, NumPy, Scikit-learn, K-Means Clustering, Matplotlib, Jupyter Notebooks.
  • This project focuses on generating an automated operation schedule for water pumps by analyzing reservoir water levels and pump start/stop data. The system uses historical measurements collected at 5-minute intervals and employs K-Means clustering and regression modeling to determine optimal pump operation times, even when real-time water level data is unavailable. Key contributions include data preprocessing, time-series analysis, clustering-based schedule generation, and visualization of daily operation plans, enabling efficient water management and reducing manual workload. 💧⚙️

Automated Ventilation

  • Used Technologies: Arduino, MQ-135 Air Sensor, Python, Online Dashboard.
  • This project aligns with the 4th Sustainable Development Goal by introducing an automated ventilation system to improve classroom air quality, thus enhancing the learning environment. Utilizing an MQ-135 air sensor and Arduino, the system monitors CO2 levels and activates ventilation when thresholds are exceeded, with fan speed dynamically adjusted based on real-time data. The setup is accessible online, and future enhancements include integrating machine learning for optimized responses. Inspired by Bogdanovica et al.'s study on CO2's impact on student well-being, this project aims to create healthier, more productive educational spaces. 🌍🏫🔧
